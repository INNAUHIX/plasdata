# 询价采购弹窗实现计划

## 任务概述
参考汽车之家"获取底价"弹窗样式，为塑库网材料详情页设计一个询价采购弹窗。点击"询价采购"按钮时打开弹窗。

## 参考样式分析

### 汽车之家弹窗特点：
1. **标题栏**：居中显示"获取底价"，右侧有关闭按钮(X)
2. **产品信息区**：左侧产品图片，右侧产品名称、指导价、参考价
3. **表单区域**：
   - 手机号码（输入框 + 占位提示）
   - 上牌城市（下拉选择）
   - 经销商（下拉选择）
   - 以旧换新（开关切换）
4. **底部**：蓝色主按钮"获取底价"、隐私政策勾选

### 塑库网表单内容（第二个截图）：
1. **顶部宣传区**：
   - 大标题："工程塑料询价慢怎么办？"
   - 副标题："塑库网快速报价服务 10分钟出精准报价，告别低效等待！"
   - 机器人形象图片
2. **表单字段**：
   - 材料型号（输入框）
   - 选择服务（单选：前期询价 / 紧急采购）
   - 联系人（输入框）
   - 联系电话（输入框）
3. **底部**：橙色主按钮"一键询价"、客服热线

## 实现方案

### 1. 创建弹窗组件
**文件**: `src/components/InquiryModal.tsx`

**组件结构**：
```
InquiryModal
├── 遮罩层 (backdrop)
├── 弹窗容器
    ├── 头部
    │   ├── 标题"获取底价"/"询价采购"
    │   └── 关闭按钮
    ├── 内容区
    │   ├── 产品信息（材料名称、价格）
    │   └── 表单
    │       ├── 材料型号输入
    │       ├── 服务类型选择（前期询价/紧急采购）
    │       ├── 联系人输入
    │       └── 联系电话输入
    └── 底部
        ├── 提交按钮"一键询价"
        └── 客服热线信息
```

**样式参考**：
- 弹窗宽度：约 420px
- 圆角：16px
- 背景：白色
- 遮罩：黑色半透明 rgba(0,0,0,0.5)
- 标题：18px 加粗
- 输入框：圆角 8px，边框 #e5e7eb
- 主按钮：橙色渐变 #f65201 → #ff8a3d
- 单选按钮：选中状态橙色

### 2. 修改 HeroBar 组件
**文件**: `src/components/HeroBar.tsx`

**修改内容**：
- 添加弹窗状态管理 `useState`
- 点击"询价采购"按钮时打开弹窗
- 导入并使用 InquiryModal 组件

### 3. 状态管理
使用 React useState 管理：
- `isOpen`: 弹窗开关状态
- `formData`: 表单数据
  - materialModel: 材料型号
  - serviceType: 服务类型（'inquiry' | 'urgent'）
  - contactName: 联系人
  - contactPhone: 联系电话

### 4. 交互逻辑
1. 点击"询价采购"按钮 → 打开弹窗
2. 点击关闭按钮或遮罩层 → 关闭弹窗
3. 表单验证：
   - 材料型号：必填
   - 服务类型：必选（默认前期询价）
   - 联系人：必填
   - 联系电话：必填，手机号格式验证
4. 点击"一键询价" → 提交表单（先 console.log 模拟）
